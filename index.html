How does ZK-PV work?
  1. 1. Install the proxy: Deploy ZK-PV as a lightweight Docker container or a managed cloud endpoint.
  2. 2. Change your LLM endpoint: Update your API URL (e.g., OpenAI) to route through ZK-PV.
  3. 3. Done! All prompts are now checked for PII and protected — GDPR-proof.
Where to install ZK-PV?
  • Directly before your LLM API (OpenAI, Anthropic, Azure, etc).
  • In a microservice workflow (protect every chat or data ingestion).
  • As a reverse proxy in your cloud or on-premise infra.
  • In the data pipeline (masking sensitive data before storage or model calls).
