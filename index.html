+ How does ZK-PV protect personal data?
ZK-PV acts as a proxy in front of your AI APIs.
It analyzes all prompts in real time and blocks or masks personal data (PII) before it can ever reach the AI or LLM provider. No modification to your app code is needed.
+ Do I need to modify my code to use ZK-PV?
Not at all. Simply change your API endpoint from api.openai.com to api.zkpv.io and insert your API key. ZK-PV takes care of the rest. Zero code change.
+ How can I verify ZK-PV’s effectiveness?
Every blocked request is logged in your dashboard, and you receive cryptographic proof of each detection. You can view a live demo or request a compliance report anytime.
